# Agent Skills — Air Pollution Project

Skills in this folder are **project-scoped**: Cursor loads them when you work in this repository. Each skill lives in its own directory with a `SKILL.md` file.

| Skill folder | Khi nào dùng |
|--------------|----------------|
| `air-pollution-overview` | Bắt đầu task mới, cần ngữ cảnh tổng thể, đạo đức dữ liệu, milestone |
| `air-pollution-data-engineering` | Preprocessing, Great Expectations, DVC, pytest data, Django models/schema |
| `air-pollution-realtime-iot` | MQTT, Mosquitto, Celery ingest, weather API, dashboard realtime |
| `air-pollution-ml-forecasting` | Prophet/XGBoost/ARIMA, TimeSeriesSplit, MLflow, Prefect |
| `air-pollution-anomaly-fastapi` | Isolation Forest, SHAP, provenance, FastAPI `/predict` `/forecast` `/explain` |
| `air-pollution-spatial-gis` | Leaflet/ngx-leaflet, Kriging, GeoPandas, spatial CV |
| `air-pollution-mlops-cicd` | GRU/LSTM, Model Registry, Evidently drift, docker-compose CI |

Chi tiết lộ trình tuần (MVP + Advanced) nằm trong `air-pollution-overview/roadmap.md`.
